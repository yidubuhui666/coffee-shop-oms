# 本次改动：导航高亮修复 + 仪表盘可点击 + 积分抵现

日期：2026-05-29　提交：e034e1d

## 1. 修复导航高亮 bug
Blueprint 重构后 `request.endpoint` 变成 `main.dashboard` 等带前缀形式，
原模板仍按旧扁平名比对，导致导航下划线高亮失效。
改为按蓝图名（`request.endpoint.split('.')[0]`）匹配，已恢复。

## 2. 仪表盘卡片可点击
仪表盘 4 张核心指标卡片改为可点击跳转：
- 今日订单 → 今日订单列表
- 今日营收 → 统计页
- 待处理订单 → 制作中订单
- 会员总数 → 会员管理
并加了悬浮上浮动效与右上角箭头提示。

## 3. 积分抵现闭环
规则：**100 积分 = ¥1**；每实付 ¥1 回赠 1 积分。

- `orders` 表新增 `points_used` 列（记录本单抵扣积分）。
- 下单页：选会员后出现“使用积分抵现”输入框，实时计算抵扣金额，含“用满”按钮；
  抵扣上限 = min(会员积分余额, 折后金额可抵上限)。
- 付款（转 PAID）：按实付金额回赠积分。
- 取消订单：退还本单抵扣的积分；若此前已付款，则收回已回赠的积分。
- 订单详情：单独列出“积分抵现”行，并显示会员当前积分。
